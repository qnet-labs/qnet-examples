Welcome to Qnet-core Examples

qnet-core is a Rust-based quantum network simulator with PyO3 bindings. This repository provides a collection of Python examples, tests, and .qnet topology fixtures that exercise the core library's capabilities.

What You Can Do​

  • Simulate entanglement distribution across multi-hop quantum networks with realistic physical constraints (decoherence, latency, fidelity decay).
  • Compare routing strategies — Lowest Latency, Highest Fidelity, and Highest Success rate — to understand trade-offs in your network design.
  • Run Monte Carlo ensembles for statistical analysis of protocol performance under randomness.
  • Load & save .qnet files — portable JSON topology definitions you can share between projects.
  • Generate standard topologies programmatically — satellite-fiber hybrids, telecom backbones, and repeater chains.
  • Explore quantum distributed computation with GHZ, cluster, and graph state protocols.

Architecture at a Glance​

qnet-core/          ← Rust core library (compiled via maturin)
    │
    ├── Qnet-coreEngine   ← simulation engine (define_network → simulate)
    ├── NodeDefinition   ← quantum node with T2 coherence lifetime
    ├── LinkDefinition   ← fiber/satellite link between nodes
    ├── StrategyType     ← routing strategy enum
    ├── generate_topology()   ← pre-built topology generators
    ├── from_qnet_file()      ← load .qnet JSON files
    └── ...

Running an Example​

from qnet-core import Qnet-coreEngine, NodeDefinition, LinkDefinition, StrategyType

engine = Qnet-coreEngine()
nodes = [
    NodeDefinition(id="Alice", memory_lifetime_t2=150.0),
    NodeDefinition(id="Bob", memory_lifetime_t2=150.0),
]
links = [
    LinkDefinition(from_node="Alice", to="Bob",
                   distance_km=25.0, base_fidelity=0.93,
                   generation_rate_hz=1000.0),
]
engine.define_network(nodes=nodes, links=links)

result = engine.request_entanglement(
    from_node="Alice", to="Bob",
    fidelity_target=0.85, max_latency_ms=200.0,
    strategy=StrategyType.HighestFidelity,
)
print(f"Success: {result.success}  Fidelity: {result.final_fidelity:.4f}")

Prerequisites​

  1. Rust toolchain (rustc, cargo)
  2. maturin — builds the PyO3 bindings
  3. Python 3.9+ with a virtual environment

See Getting Started for detailed build instructions.
