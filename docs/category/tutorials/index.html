Tutorials

Step-by-step guides for qnet-core

📄️Tutorial 1 — Routing Strategy Trade-offs (Latency vs. Fidelity)

This tutorial demonstrates how different routing strategies produce very different results on the same network. We build a simple 3-node repeater chain and run two strategies side by side.

📄️Tutorial 2 — Physical Constraint Boundaries

This tutorial shows how the simulator enforces fundamental physics — decoherence (T2 memory lifetime) and latency (speed of light in fiber). We simulate a transatlantic link with intentionally poor hardware to see constraint failures in action.

📄️Tutorial 3 — Latency-Optimized Routing (Monte Carlo Ensemble)

This tutorial shows how to run a Monte Carlo ensemble (simulate()) instead of a single request, getting statistical insights across many runs.

📄️Tutorial 4 — Highest Fidelity Routing (Purification Loops)

This tutorial shows how the HighestFidelity strategy triggers purification (distillation) cycles to achieve high-quality entanglement across a multi-hop network.

📄️Tutorial 5 — Physical Constraints Failure (Stress Testing)

This tutorial stress-tests the simulator with impossible physics — a single direct link over 4,500 km with a 2 ms latency target. We verify that the simulator correctly rejects physically impossible requests.

📄️Tutorial 6 — Sensitivity Analysis

This tutorial shows how to sweep a parameter (fidelity target) across multiple batches to understand where your network's capabilities break down. This is the quantum equivalent of an IV curve in electronics — it maps performance boundaries.

📄️Tutorial 7 — Custom Simulation Configuration

This tutorial shows how to run simulations with the default physics parameters. For full customization, see the API Reference for the SimulationConfig class.

📄️Tutorial 8 — Pre-Built Topologies

This tutorial demonstrates the generatetopology() and comparetopologies() helpers that let you build and compare network topologies without writing out every node and link manually.

📄️Tutorial 9 — Loading .qnet Files

This tutorial shows how to load network topologies from .qnet JSON files instead of defining them programmatically. This is useful for sharing, version-controlling, and reusing network configurations.

📄️Tutorial 10 — Building .qnet Files Programmatically

This tutorial shows how to construct a .qnet file entirely from Python code, save it to disk, validate it, reload it, and simulate on it — all in one script.

📄️Tutorial 11 — Strategy Battle (All Three Strategies Compared)

This tutorial runs all three routing strategies on the exact same network and compares results side by side. It's the best way to understand the trade-offs between speed, fidelity, and robustness.

📄️Tutorial 12 — Multi-Path Routing on a Mesh Topology

This tutorial demonstrates how different routing strategies choose different paths when multiple routes are available between two endpoints.

📄️Tutorial 13 — Seed Determinism (Reproducible Simulations)

This tutorial demonstrates that the Monte Carlo simulator is fully deterministic when given a seed, and non-deterministic without one. This is essential for reproducible experiments and fair configuration comparisons.

📄️Tutorial 14 — Link Utilization Heatmaps

This tutorial shows how to inspect which links are actually used during simulation and identify bottlenecks in your network using the linkutilizationheatmap field.

📄️Tutorial 15 — Distributed Quantum Computation

This tutorial covers the distributed_computation module, which simulates multi-party quantum computation protocols over a quantum network. You can run distributed computations with GHZ, cluster, or graph states across multiple nodes.
