.qnet File I/O API Reference

Functions​

from_qnet_file(path)​

Load a .qnet JSON file and return a ready-to-simulate Qnet-coreEngine.

from qnet-core import from_qnet_file
engine = from_qnet_file("network.qnet")
ParameterTypeDescription
pathstrPath to the .qnet JSON file

Returns: Qnet-coreEngine — pre-configured with the network from the file.

validate(path)​

Validate a .qnet file for structural correctness.

from qnet-core import validate
result = validate("network.qnet")
ParameterTypeDescription
pathstrPath to the .qnet JSON file

Returns: dict with keys:

KeyTypeDescription
validboolWhether the file is structurally valid
errorsList[str]List of validation errors (empty if valid)
warningsList[str]List of warnings (e.g., null optional fields)

Example:

result = validate("network.qnet")
if not result["valid"]:
    for error in result["errors"]:
        print(f"  Error: {error}")

diff(path_a, path_b)​

Compare two .qnet files and return a structural diff.

from qnet-core import diff
result = diff("network_v1.qnet", "network_v2.qnet")
ParameterTypeDescription
path_astrPath to the first .qnet file (baseline)
path_bstrPath to the second .qnet file (compared against baseline)

Returns: dict with keys:

KeyTypeDescription
nodes_addedList[str]Nodes in B but not in A
nodes_removedList[str]Nodes in A but not in B
links_addedList[str]Links in B but not in A
links_removedList[str]Links in A but not in B
summarystrHuman-readable summary of differences

Classes​

PyQnet-coreFile(name)​

Build a .qnet file programmatically.

from qnet-core import PyQnet-coreFile, PyQnet-coreNodeType, PyQnet-coreLinkType

f = PyQnet-coreFile("my_network")
f.metadata.description = "My custom network"
f.metadata.author = "you"

f.add_node("Alice", memory_lifetime_ms=150.0, node_type=PyQnet-coreNodeType.Ground)
f.add_node("Bob", memory_lifetime_ms=150.0, node_type=PyQnet-coreNodeType.Ground)
f.add_link("", "Alice", "Bob", 25.0, 0.93, 1000.0)

f.config = PyQnet-coreConfig(alpha_loss=0.045, gamma_swapping=0.92)
f.constraints = PyQnet-coreConstraints(fidelity_target=0.85, max_latency_ms=500.0)

f.save("my_network.qnet")
MethodParametersDescription
add_node(id, ...)(id, memory_lifetime_ms?, memory_capacity?, node_type?)Add a node to the network
add_link(id, from, to, ...)(id, from_node, to, distance_km, base_fidelity, generation_rate_hz, link_type?, satellite?)Add a link between nodes
save(path)(path)Write the file to disk as JSON

ValueDescription

PyQnet-coreLinkType Enum​

ValueDescription
FiberOptical fiber link
SatelliteFree-space / satellite link

PyQnet-coreSatelliteExtension(visibility, weather_factor)​

Satellite-specific conditions. The effective generation rate is base_rate × visibility × weather_factor.

ParameterTypeDescription
visibilityfloatSignal visibility (0.0–1.0)
weather_factorfloatWeather attenuation (0.0–1.0)

Method: effective_rate(base_rate) — returns base_rate × visibility × weather_factor.

PyQnet-coreConfig(alpha_loss, beta_fidelity_decay?, gamma_swapping, max_attempts?)​

Global simulation configuration.

ParameterTypeDefaultDescription
alpha_lossfloatrequiredFiber attenuation coefficient (dB/km)
beta_fidelity_decayfloat0.02Fidelity decay per km
gamma_swappingfloat0.92Entanglement swapping efficiency
max_attemptsint100Max retries before giving up

PyQnet-coreConstraints(fidelity_target, max_latency_ms)​

Global simulation constraints applied to all requests on this network.

ParameterTypeDescription
fidelity_targetfloatDefault fidelity target (0.0–1.0)
max_latency_msfloatDefault max latency in ms
